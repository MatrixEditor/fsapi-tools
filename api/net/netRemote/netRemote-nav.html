This node returns the list of available items in the attached storage.

    It is an enum field and accepts the following values: - 0: “Directory” - 1: “PlayableItem” - 2: “SearchDirectory” - 3: “Unknown” - 4: “FormItem” - 5: “MessageItem” - 6: “AmazonLogin” - 7: “FetchErrItem”

    This field is also an enum field: - 0: “None” - 1: “Station” - 2: “Podcast” - 3: “Track” - 4: “Text” - 5: “Password” - 6: “Options” - 7: “Submit” - 8: “Button” - 9: “Disabled”

  • graphicUri: TODO
